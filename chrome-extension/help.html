Protection Rules

保护规则

内置脱敏规则。当前不支持自定义规则。

严重

严重OpenAI Key

会泄漏 AI 服务访问额度或项目权限。

OPENAI_API_KEY=[TOKEN_VALUE]
内置启用

严重Bearer 访问令牌

会泄漏账号或服务访问权限。

Authorization: Bearer [TOKEN_VALUE]
内置启用

严重Cookie sessionid

会泄漏登录会话或身份状态。

Cookie: sessionid=[SESSION_VALUE]
内置启用

严重URL 参数 token/api_key

会泄漏请求里的密钥、访问口令或临时凭证。

?api_key=[TOKEN_VALUE]
内置启用

严重GitHub Token / AWS Key / Stripe Key

会泄漏代码仓库、云资源或支付服务访问权限。

ghp_[TOKEN_VALUE]
内置启用

严重数据库 URL / PEM 私钥

会泄漏数据库连接信息或私钥材料。

DATABASE_URL=postgres://[SECRET]
内置启用

提醒

提醒Sentry DSN

可能暴露项目上报入口。

SENTRY_DSN=https://[PUBLIC]:[SECRET]@host/id
内置启用

提醒Webhook URL

可能暴露 Slack、Discord 等 webhook 投递入口。

WEBHOOK_URL=https://hooks.slack.com/[PATH]
内置启用

提醒URL userinfo / Basic Auth

可能把用户名、密码或基础认证值带入 AI。

https://user:[PASSWORD]@host
内置启用

提醒npm / pypirc / Docker registry auth

可能泄漏包管理器或镜像仓库认证信息。

_authToken=[TOKEN_VALUE]
内置启用

Redaction

脱敏设置

设置发现敏感信息后的处理方式。

Privacy

隐私与存储

VeilPaste 的原则是本地处理、最小存储。

  • 不上传粘贴内容
  • 不保存敏感值
  • 不保存完整 prompt
  • 不发网络请求
  • 只记住你是否开启“以后自动脱敏”。
  • 设置页只在你点击设置时打开。
  • 只在 ChatGPT、Claude、Perplexity、豆包和 Qwen 上工作。
  • 临时记录刷新页面后失效。
