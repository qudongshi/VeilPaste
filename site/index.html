Local paste guard for AI workflows

在粘贴给 AI 之前，先本地发现明显的密钥风险。

VeilPaste 是一个小型开发者工具。它在 ChatGPT、Claude、Perplexity、豆包和 Qwen 上监听用户主动粘贴的文本，发现明显的 API key、Bearer token、Cookie、 数据库连接串或私钥片段后，先提示你选择是否脱敏。

查看隐私政策 获取支持

本地处理

检测与脱敏在浏览器本地完成。当前 Chrome 插件不上传粘贴内容，不保存敏感值，也不发网络请求。

只在指定 AI 网站运行

当前保护范围限定为 ChatGPT、Claude、Perplexity、豆包和 Qwen，避免请求过宽的网站权限。

只处理粘贴动作

插件只监听用户触发的 paste 事件，不扫描整个页面，不拦截发送、提交或点击动作。

当前可识别的常见风险

  • Authorization Bearer token 和 Basic Auth 凭证
  • OpenAI、AWS、GitHub、Stripe 等常见访问密钥
  • Cookie 会话值、URL 查询参数中的 token/key/secret/password
  • 数据库 URL、Redis URL、MongoDB URI、Sentry DSN 和 webhook URL
  • PEM 私钥、npm token、pypirc 凭证、Docker registry auth

当前边界

VeilPaste 只能帮助发现明显的开发者密钥和凭证片段，不能保证一段 prompt 已经安全，也不能替代人工审查。处理真实生产数据前，请先确认你的团队安全要求。

当前公开版本不是账号或订阅产品，不包含团队管理、云端同步、遥测、远程规则包或付费授权逻辑。
